If you have Vim
  then ...
---------------------------------------------------------------
You must add in your .vimrc:
	au BufRead,BufNewFile *.txt setfiletype eathenascript
---------------------------------------------------------------
This means that our syntax file will be apply to all files with
the extension txt
---------------------------------------------------------------
If you do not use "pathogen", then we must 
put files in syntax and ftpplugin folders
---------------------------------------------------------------
To use autocomplite we must key in insert mod  <c-x><c-o>
---------------------------------------------------------------
To use template we must key in insert mod >ft and >nt
---------------------------------------------------------------
---------------------------------------------------------------
You can may take documentation about any special words or  function
or simply introduction in eAthena script simply using the command :h ea-<word>

>example
:h ea-syntax
